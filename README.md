# MiGoi Membership Monthly Payment

## Features

- Use ETH/USDT/DAI/USDC/LINK token to purchase membership monthly.
- Based on zkSync Checkout
- Using Chainlink price feeds
- Confirm tx then set user to membership. (1/2)
